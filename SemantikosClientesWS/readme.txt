Clientes WS para Semantikos
===========================

Este directorio alberga el codigo fuente para consumir los servicios web para tres tecnologías distintas:
   - Java
   - .NET
   - PHP

Cada aplicacion provee:
   - Una página HTML para invocar el cliente en la tecnología seleccionada que invoca el WS.
   - Una componente en la tecnología que realiza la invocación y recupera el resultado y refrezca en la página.
